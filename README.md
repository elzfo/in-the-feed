## üéÆ Play the Game

This is a web-based game prototype built with React and Tailwind CSS.

### **How to Play**
1. **Objective**: Survive your 3-minute shift as a content moderator during a high-stakes election.
2. **Controls**:
   - **Swipe/Click Red Posts**: Remove misinformation and viral fake news.
   - **Ignore Green/Neutral Posts**: Let legitimate news flow.
   - **Watch Your Energy**: Removing posts costs energy. Don't burn out!
3. **Losing Condition**: If too much misinformation slips through, the "Toxicity" meter fills up, and the system collapses.

## üõ†Ô∏è Project Setup (For Beginners)

If you want to run this game on your own computer, follow these steps:

### **Prerequisites**
You need to have **Node.js** installed.
1. Download and install Node.js from [nodejs.org](https://nodejs.org/).
2. Verify installation by opening a terminal (Command Prompt on Windows, Terminal on Mac) and typing:
   ```bash
   node -v
   ```

### **Installation**
1. **Clone the repository**:
   ```bash
   git clone https://github.com/elzfo/in-the-feed.git
   cd in-the-feed
   ```

2. **Install dependencies**:
   ```bash
   npm install
   # OR if you use pnpm
   pnpm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   # OR
   pnpm dev
   ```

4. Open your browser and go to `http://localhost:5173` (or the port shown in your terminal).

## üé® Design Philosophy
The game uses a **Cyberpunk / Glitch Art** aesthetic to represent the degradation of information quality.
- **Visuals**: CRT scanlines, glitch effects, and high-contrast neon colors.
- **Metaphor**: The "Feed" is an overwhelming torrent of data. The player is fighting a losing battle against an algorithm designed for engagement, not truth.
